# Game Assets

This directory contains all game assets including audio, sprites, and fonts.

## Directory Structure

- **`audio/`** - Sound effects and music files
- **`fonts/`** - Typography files for UI display
- **`generated/`** - Runtime-generated content (e.g., procedural textures)
- **`sprites/`** - Game graphics including UI elements, icons, and animations

## Asset Organization

Assets are organized by type and functional purpose:
- Game graphics in `sprites/`
- Sound resources in `audio/`
- Text display in `fonts/`
- Dynamic content in `generated/`

## Usage

These assets are loaded by Godot scene files and referenced in the project configuration.
