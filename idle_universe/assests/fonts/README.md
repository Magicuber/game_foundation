# Font Assets

This directory contains typography files for the game's UI.

## Contents

- `JetBrainsMono-Regular.ttf` - Monospace font for displaying numeric values and scientific notation

## Usage

Fonts are used in:
- Resource counters displaying DigitMaster values
- UI elements showing element names
- Scientific notation displays
- Code-like text elements

## Import

Font `.import` files contain Godot's import settings for the font.
