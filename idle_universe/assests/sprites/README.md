# Sprite Assets

This directory contains all game graphics including element sprites, UI elements, and animations.

## Contents

### Element Sprites

- `Element1.png` through `Element119.png` - Individual element sprites for atomic numbers 1-119
- `elements_01_strip119.png` - Sprite strip containing all elements
- `elements_foil_strip119.png` - Foil variant of elements
- `elements_holo_strip119.png` - Holographic variant of elements
- `elements_poly_strip119.png` - Polychrome variant of elements

### UI Elements

- `header.png` - Header/banner graphics
- `spr_BUTTON.png` - Button sprites
- `spr_menu_btn.png` - Menu button
- `spr_close_btn1.png` - Close button
- `spr_next_btn.png` / `spr_prev_btn.png` - Navigation buttons
- `spr_shop_btn.png` - Shop button
- `spr_upgrade_btn.png` - Upgrade button
- `spr_zin_btn.png` / `spr_zout_btn.png` - Zoom buttons
- `spr_nav_btns_strip6.png` - Combined navigation buttons
- `spr_eleupgds_background.png` - Element upgrades background

### Planet Graphics

- `spr_planet_A_001.png` - Planet A sprite
- `spr_planet_B_001.png` - Planet B sprite
- `planet_A_split25.png` - Planet A level progression sprites
- `atoms_menu_strip119.png` - Atomic menu sprites

### Other Sprites

- `MSTR_Sprite_sheet.png` - Master sprite sheet
- `spr_smasher.png` - Particle smasher animation
- `spr_t1_upgrd.png` / `spr_t2_upgrd.png` - Upgrade tier graphics
- `spr_era_strip4.png` - Era selection graphics

## Import Files

Each `.png` has a corresponding `.png.import` file generated by Godot storing import settings.
