# Audio Assets

This directory contains all sound effects and music files for the game.

## Contents

Audio files for game including:
- Sound effects for UI interactions
- Element collection sounds
- Particle smasher effects
- Level up sounds
- Ambient music (if any)

## Usage

Audio is referenced by scene files and played through Godot's AudioStreamPlayer nodes.

## Import

Audio files should be imported through Godot's import system for optimal format and compression.
