# Generated Assets

This directory contains assets that are generated or processed at runtime.

## Contents

- `profile_icon.png` - Runtime-generated profile icon

## Purpose

This directory stores assets created by the game or build process:
- Cache files
- Profile pictures
- Runtime-generated textures
- Processed image data

## Notes

Files in this directory may be regenerated automatically and should not be manually edited or version-controlled (except for templates).
