# Idle Universe

Idle Universe is a Godot-based idle game where players progress through the periodic table of elements, unlocking new elements and planets while collecting dust. The game features a complex progression system with elements, upgrades, blessings, planets, and oblations.

## Project Structure

- `addons/` - Godot addon plugins including Aseprite importer
- `assests/` - Game assets including sprites, audio, and fonts
- `docs/` - Project documentation
- `scenes/` - Godot scene files
- `src/` - Source code organized by functionality

## Core Systems

1. **Game State Management** - Centralized game state with `GameState` class
2. **Element Progression** - Element unlocking and progression system
3. **Planet Management** - Planet purchase, ownership, and progression
4. **Upgrade System** - Various upgrades that enhance gameplay
5. **Blessing System** - Random blessings that provide bonuses
6. **Oblation System** - Dust recipes and oblation mechanics
7. **Save System** - Persistent game state saving and loading

## Getting Started

This is a Godot 4.x project. Open with the Godot editor to run and develop.
