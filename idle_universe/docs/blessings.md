# Blessings Documentation

## Overview

The blessing system provides random permanent bonuses earned through gameplay. Players generate "blessing mass" by producing elements, then spend that mass to roll random blessings from different rarities.

**Key Files:**
- [`src/core/managers/blessing_manager.gd`](../src/core/managers/blessing_manager.gd) - Blessing logic
- [`src/core/state/blessing_state.gd`](../src/core/state/blessing_state.gd) - Blessing state class
- [`src/core/state/blessing_rarity.gd`](../src/core/state/blessing_rarity.gd) - Rarity definitions
- [`src/data/blessings.json`](../src/data/blessings.json) - Blessing definitions

## Blessing Structure

```gdscript
class_name BlessingState

var id: String           # Unique identifier
var name: String         # Display name
var description: String  # Effect description
var rarity: String       # Rarity tier
var level: int          # Current level (0 = undiscovered)

var effect_type: String  # Type of effect
var effect_base: float   # Base effect value
var effect_scaling: float  # Bonus per level
var placeholder: bool    # Whether this is placeholder content

func get_effect_value() -> float:
    if level <= 0:
        return 0.0
    return level × effect_base
```

### Roll Chance Calculation

```gdscript
func roll_blessing_rarity() -> String:
    var total_weight := 0.0
    var rollable_rarities := get_rollable_rarities()
    
    for rarity in rollable_rarities:
        total_weight += blessing_rarity_roll_weights[rarity]
    
    var roll := random_float() × total_weight
    var cursor := 0.0
    
    for rarity in rollable_rarities:
        cursor += blessing_rarity_roll_weights[rarity]
        if roll <= cursor:
            return rarity
```

## Blessing Generation

### Mass Accumulation

Blessing mass is generated by producing elements:

```gdscript
func apply_blessing_progress_for_generated_element(
    element: ElementState, 
    amount: DigitMaster
) -> void:
    # Skip element 0 (Protons) - no value
    if element.index <= 0:
        return
    
    # Mass = amount × element_index
    var generated_mass := amount.multiply_scalar(float(element.index))
    blessings_progress_mass = blessings_progress_mass.add(generated_mass)
    
    # Check if we can award a blessing
    while blessings_progress_mass.compare(get_next_blessing_cost()) >= 0:
        var next_cost := get_next_blessing_cost()
        blessings_progress_mass = blessings_progress_mass.subtract(next_cost)
        blessings_count += 1
        unopened_blessings_count += 1
```

### Cost Formula

Blessing costs increase quadratically:

```gdscript
# Formula: cost = 10x² + 400x + 1600
# where x = blessing_count

func get_next_blessing_cost() -> DigitMaster:
    var x := float(blessings_count)
    var cost := 10.0 × x × x + 400.0 × x + 1600.0
    return DigitMaster.new(cost)
```

**Example costs:**
| Blessing # | Cost (mass) |
|------------|-------------|
| 1 | 11,010 |
| 2 | 11,244 |
| 5 | 12,450 |
| 10 | 14,600 |
| 50 | 46,050 |

### Rolling a Blessing

```gdscript
func roll_random_blessing_id() -> String:
    var rarity := roll_blessing_rarity()
    var available := get_rollable_blessing_ids_for_rarity(rarity)
    
    if available.is_empty():
        return ""
    
    var index := random_range(0, available.size() - 1)
    return available[index]

func award_random_blessing() -> void:
    var blessing_id := roll_random_blessing_id()
    if blessing_id.is_empty():
        return
    
    var blessing = get_blessing_state(blessing_id)
    if blessing == null or blessing.placeholder:
        return
    
    blessing.level += 1
    invalidate_blessing_effect_cache()
```

## Opening Blessings

Players accumulate unopened blessings, then open them to roll effects:

```gdscript
func open_earned_blessings() -> int:
    var count := unopened_blessings_count
    if count <= 0:
        return 0
    
    for _i in range(count):
        award_random_blessing()
    
    unopened_blessings_count = 0
    return count
```

## Effect Totals

Blessing effects stack with upgrade effects. The manager provides cached totals:

```gdscript
func get_blessing_effect_total(effect_type: String) -> float:
    ensure_blessing_effect_cache()
    return _cached_blessing_effect_totals.get(effect_type, 0.0)

func ensure_blessing_effect_cache() -> void:
    if not _blessing_effect_cache_dirty:
        return
    
    _cached_blessing_effect_totals.clear()
    for blessing_id in blessing_ids_in_order:
        var blessing = get_blessing_state(blessing_id)
        if blessing.level > 0:
            var current := _cached_blessing_effect_totals.get(blessing.effect_type, 0.0)
            _cached_blessing_effect_totals[blessing.effect_type] = current + blessing.get_effect_value()
```

## Blessing Reset

Players can optionally reset blessings (usually tied to prestige):

```gdscript
func reset_blessings() -> bool:
    var changed := reset_blessing_levels_to_zero()
    unopened_blessings_count = blessings_count
    
    if changed:
        invalidate_blessing_effect_cache()
    
    return changed

func reset_blessing_levels_to_zero() -> bool:
    for blessing_id in blessing_ids_in_order:
        var blessing = get_blessing_state(blessing_id)
        if blessing.level > 0:
            blessing.level = 0
            changed = true
    return changed
```

## Save Integration

Blessings persist through saves:

```gdscript
# Saving
var blessing_data = {}
for blessing_id in blessing_ids_in_order:
    var blessing = get_blessing_state(blessing_id)
    blessing_data[blessing_id] = {
        "level": blessing.level
    }

# Loading
func apply_saved_blessing_levels(saved_blessings: Dictionary) -> void:
    for blessing_id in saved_blessings.keys():
        var blessing = get_blessing_state(blessing_id)
        if blessing:
            var data = saved_blessings[blessing_id]
            blessing.level = data.get("level", 0)
```
